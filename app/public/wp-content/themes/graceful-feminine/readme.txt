== Description ==

Graceful Feminine is a free feminine WordPress Feminine theme. Its perfect for any kind of blog or website like lifestyle, fashion, travel, tech, health, fitness, beauty, food, news, magazine, feminineging, personal, professional, etc. Its fully Responsive and Retina Display ready, clean, modern and minimal design. Graceful Feminine is WooCommerce compatible, supports RTL(Right To Left) and is optimized for SEO.

== Frequently Asked Questions ==

= How to install Graceful Feminine theme =
1. Login your WordPress dashboard.
2. Go to "Appearance" -> "Theme" and click on "Add New" button.
3. Click on "Upload Theme" button and choose the theme's zip file you have downloaded.
4. Click on "Install Now" button.
5. Activate the theme.
6. Now go to "Appearance" -> "Customize" to configure theme options.
